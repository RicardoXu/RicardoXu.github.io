Markdown 使用指南

标题

在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加 # 号即可。

一级标题

二级标题

三级标题

以此类推，总共六级标题，建议在井号后加一个空格，这是最标准的 Markdown 语法。

另外，如果要在第一个字符打出 # 本身这个符号，在其前面加一个空格即可。

列表

加 * 即可变为无序列表，加 1. 2. 3. 可变为有序列表。

符号要和文字之间加上一个字符的空格。

  • a
  • b
  • c

两个列表不能相邻。

  1. a
  2. b
  3. c

在需要被嵌套的列表第一个符号前加空格可以实现嵌套。

引用

引用格式。

Example

在文本前加入 > 这种尖括号即可。

图片与链接

插入链接与插入图片的语法类似，区别是图片的话需要在最前面加入一个 ! 号。

[name](url)

粗体与斜体

用一对 * 号括起来的部分是可以转换成加斜/加粗的。

一对表示加斜，两对表示加粗，三对表示加斜加粗。

代码框

一对 ` 可以实现代码框的功能。

分割线

三个星号 * 可以添加一道分割线。

创建链接

<>

为输入的URL或邮箱自动创建链接，如test@email.com。

转义字符

转义字符 \ 在 Markdown 里是生效的。

使用平台

Windows: MarkdownPad 2

Linux： Ubuntu ？

结束语

基本功能介绍完毕，其他功能以后视需要进行补充。
