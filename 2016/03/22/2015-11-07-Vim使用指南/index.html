Vim 使用指南

OS: Ubuntu

安装： sudo apt-get install vim-gtk

配置： sudo vim /etc/vim/vimrc

在最后一行开始输入：

set nu // 在左侧行号

set tabstop=4 //tab 长度设置为 4

set nobackup //覆盖文件时不备份

set cursorline //突出显示当前行

set ruler //在右下角显示光标位置的状态行

set autoindent //自动缩进

Insert 模式

按下 i 键进入 Insert 模式，在 Insert 模式中，可以进行文字的编辑。

按下 Esc 键返回 Normal 模式。

Normal 模式

基本操作

  • x 删当前光标所在的个字符。
  • :wq 存盘 + 退出(:w 存盘, :q 退出)。
  • dd 删除当前行，并把删除的行存到剪贴板里。
  • p 粘贴剪贴板到当前行的下一行
