MySQL指令小结

发表于 2016-11-07 |

这里收录了一些常用的MySQL指令。

开启/关闭MySQL服务：

net start mysql
net stop mysql

登录：

mysql -u root -p

新建数据库操作：

create database 数据库名 character set utf8;
use 数据库名;
drop database 数据库名;

新建表操作：

create table students
(
    id int unsigned not null auto_increment primary key,
    name char(8) not null,
    sex char(4) not null,
    age tinyint unsigned not null,
    tel char(13) null default "-"
);

常用查改增删：

select * from 表名;
update 表名 set 列名 = ? where 列名 = ?;
insert into 表名 (列名1, 列名2, 列名3) values(?, ?, ?);
delete from 表名 where 表达式;
drop table <表名>;

查看表的信息

show full columns from 表名;

修改列的属性

alter table 表名 change 列名 列名 修改成的目标类型;

重设root用户密码：

set password for root@localhost = password('123456');

设置默认的字符编码：

// MySQL
show variable like '%character%' // 查看当前字符编码

// bash
service mysql stop  
cd /etc/mysql/ // 默认的安装路径
vim my.cnf 

// vim,在[mysqld]下增加如下两行设置:
character_set_server=utf8  
init_connect='SET NAMES utf8'  

// bash
service mysql start
# MySQL
